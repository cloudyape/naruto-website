You can Auto Write any HTML Code via AI in any component

If you want to auto write HTML code using AI, you just need an OpenAPI key and to follow Below steps
  • In the Python Terminal Run command pip install openai
  • Now replace apikey.py with your OpenAI key
  • Now in your component place variable like syntax -> @@variableName@@ -> Eg @@hello@@. AI will replace the this place with autogenerated code
  • Now in naruto terminal write code -> naruto auto create variableName. Eg naruto auto create hello
  • It will ask for component. Enter the component you would like to replace with auto code
  • It will then ask for prompt. Just Enter prompt of the code @@hello@@ will be replaced with an auto generated codes
