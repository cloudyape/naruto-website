Download Naruto from Below

Download/Clone Naruto from Github

Once Downloaded, Follow the steps Below

  • Extract Naruto Zip File
  • Run commands.py using command -> python command.py
  • Once you enter in Naruto Exclusive Terminal Run -> naruto setup new to create new File
  • Lastly Do - naruto run to run server on port 8000 or naruto run port portnumber to run on custom port. Eg naruto run port 6789
